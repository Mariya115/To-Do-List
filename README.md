# 📝 Task Manager App

A full-stack Task Manager application that allows users to efficiently create, manage, and track their daily tasks through a simple and responsive interface.

---

## 🚀 Features

- Create, update, and delete tasks (CRUD operations)
- Mark tasks as completed or pending
- Responsive and user-friendly interface
- Organized task tracking for better productivity
